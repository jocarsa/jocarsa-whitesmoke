La manera más fácil de gestionar tus currículums

Crea, edita y comparte CVs profesionales con un sistema multiusuario intuitivo.

Características destacadas

Edición en tiempo real

Modifica tu currículum de forma inmediata y visualiza los cambios instantáneamente en la vista previa.

Sistema multiusuario

Cada usuario gestiona su propio CV, almacenado de forma segura e independiente en la base de datos.

Almacenamiento en JSON

La información de cada usuario se guarda como JSON, brindando flexibilidad y compatibilidad con numerosas aplicaciones.

Diseño personalizable

La plantilla de currículum ofrece una estructura clara, pero puedes adaptarla según tus necesidades y estilo.

Seguridad de acceso

Inicia sesión y protege tus datos con credenciales únicas. Cada usuario solo accede a su propia información.

Fácil integración

Gracias a la API de guardado, es muy sencillo integrar este sistema con otras plataformas o servicios de recursos humanos.

Beneficios y Ventajas

1. **Ahorro de tiempo**: El editor en línea y la previsualización inmediata permiten crear un CV profesional en minutos.

2. **Centralizado**: Almacena toda tu información de forma organizada, evitando archivos dispersos y confusiones de versiones.

3. **Accesible**: Ingresa desde cualquier dispositivo con conexión a internet y mantén tu currículum siempre actualizado.

4. **Colaboración**: Comparte tu CV con empleadores o reclutadores de forma directa, o genera un PDF en cualquier momento.

Características Técnicas

El sistema está construido con PHP y SQLite, lo que lo hace ligero y fácil de desplegar. Cada cuenta de usuario se guarda en la base de datos con su propio CV, mientras que el editor dinamiza la información en tiempo real.

1. **Base de datos SQLite**: Rápida y sin complicaciones de configuración.

2. **Estructura JSON**: Actualiza, añade y elimina secciones de tu CV con gran flexibilidad.

3. **Motor de plantillas**: La previsualización se basa en un template personalizable que extrae tus datos dinámicamente.

4. **Seguridad con sesiones**: Cada usuario inicia sesión y solo puede ver y editar sus propios datos.

Gracias a estas funciones, Jocarsa Resume Manager se adapta a freelancers, estudiantes y profesionales que deseen tener siempre un CV actualizado.
